Inférence de type Hindley Milner petit à petit

Le but de cet article est de décrire et expliquer l'algorithme W, d'inférence de type dans le système de type Hindley-Milner. Les descriptions seront accompagnées de code en OCaml, pour faliciter l'implémentation. Premièrement, on doit définir le problème que l'on essaye de résoudre. Le langage que auquel on s'intéresse est un lambda calculus augmenté d'entier et de structures let. Ainsi, au format BNF, on a la syntaxe suivante :
e, e' ::= e e' application
| let v = e in e' définition
| λv.e abstraction
| n entier
On cherche une fonction inference, qui prend en argument une expression et renvoie le type de cette-dernière. On commence par le cas le plus simple, l'entier.
